# Road-Accident-Analytics
Road accident data analysis for identifying accident patterns and causes
## Project Overview
This project focuses on analyzing road accident data to understand
accident causes, high-risk locations, and time-based patterns.

## Objectives
- Identify accident-prone areas
- Analyze accidents by time, weather, and vehicle type
- Support road safety improvement decisions

## Tools Used
- Microsoft Excel
- Power BI
- SQL

## Key Insights
- Accidents peak during evening hours
- Certain locations show repeated accident patterns
- Two-wheelers involved in higher accident rates

## Outcome
The analysis provides actionable insights to improve road safety
and reduce accident occurrences.
